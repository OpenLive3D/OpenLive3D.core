# OpenLive3D Core

The core will have these modules:
 - Camera Control
 - ML Module
 - Motion Convertion
 - Interpolator
 - Thread Control

## Exported Class:
 - OL3DCore
   - Core Function
     - initCore()
     - getVRMMovement()
   - Logs
     - getCoreLog()
     - getPoI()
   - Camera
     - setVideoStream(device_id)
     - getCameraFrame()
