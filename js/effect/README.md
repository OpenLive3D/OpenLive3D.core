# Effect

Effect manager handles the canvas effects.

 - List Effects
 - Return Functions to Apply Effects

## Add Effect Process

 - Add New Effect File with Functions and Parameters:
   - enableEffect
   - disableEffect
   - updateEffect
   - parameters
 - Add Effect File to:
   - `index.html`
   - `index-mobile.html` (web-version)
 - Add Effect Functions and Parameters to:
   - `effect-manager.js`
